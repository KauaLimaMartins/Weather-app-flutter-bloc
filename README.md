## How to run

1. First clone the repo:

```bash
git clone https://github.com/KauaLimaMartins/Weather-app-flutter-bloc.git
```

<br />

2. In the repo path:

```bash
flutter pub get
```

<br />

3. And finally:

```bash
flutter run
```
